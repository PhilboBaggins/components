Pinout
------

| Pin number | Pin name | Description |
| ---------- | -------- | ----------- |
|     1      |   DAT    | Data        |
|     2      |   CMD    | Command     |
|     3      |   N/C    | Vibrations  |
|     4      |   GND    | Ground      |
|     5      |   VCC    | Power       |
|     6      |   ATT    | Attention   |
|     7      |   CLK    | Clock       |
|     8      |   N/C    | Unknown     |
|     9      |   ACK    | Acknowledge |
